#PR.9-Visualization
Aim

To develop a menu driven Python program for performing basic data analysis and visualization using CSV dataset.

Requirements
Python 3.x
pandas
matplotlib
Description

This project is a simple data analysis and visualization program. The user can load a dataset and perform different operations such as exploring data, handling missing values, generating statistics and creating graphs. The program is menu driven and easy to use.

Features
Load CSV dataset
Explore data (head, tail, columns, info)
Handle missing values
Generate descriptive statistics
Data visualization:
Bar Plot
Line Plot
Scatter Plot
Pie Chart
Histogram
Save visualization

Conclusion

This project helps in understanding basic data analysis and visualization using Python libraries like pandas and matplotlib.
